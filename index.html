Hi, I'm Peter.

Software Engineering

I build web and mobile apps and have experience with all phases of the a project's lifespan. I've been building web applications for ten years and have background in Javascript, HTML, CSS, Django, PHP, and MySQL along with numerous libraries and plugins for these tools. Most recently, I've taken a strong interest in AngularJS.

Blogging

I have been blogging about video games in my spare time for the past five years. I've managed two personal blogs and contributed to others. I've written primarily about the critical analysis of video games and review those that I've played myself. I believe that writing is an important aspect of being a better programmer.

Game Development

In addition to writing software for the web and mobile devices, I write code that serves no other purpose than to amuse people. Video games are what initially brought my attention to programming and software engineering, and continuing to program games gives me an opportunity to appreciate how people interface with technology.
